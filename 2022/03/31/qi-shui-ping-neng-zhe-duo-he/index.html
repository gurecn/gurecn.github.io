#汽水瓶#能者多喝
模拟 算法

阅读时长:   1 分

描述

某商店规定：三个空汽水瓶可以换一瓶汽水，允许向老板借空汽水瓶（但是必须要归还）。小张手上有n个空汽水瓶，她想知道自己最多可以喝到多少瓶汽水。

HJ22汽水瓶

思路

递归调用：
最简单的思路，首先拿着n个瓶子全部换成汽水，能换成n/3瓶汽水，并且手里剩余n%3个瓶子，全部喝完后手里会有n/3 +n%3个汽水瓶 。拿着这些汽水瓶又可以重新换汽水，因为可以借瓶子，所以最后手里汽水瓶数量为2时，可以再换一瓶，汽水瓶数量小于2时，就喝水完成。
通过上述方式，使用递归调用进行实现。
懒人方式
我们可以找规律：

瓶子数量 喝水数量
0 0
1 0
2 1
3 1
4 2
5 2
6 3
7 3
8 4
9 4

发现一个规律，空瓶的数量n/2就是能喝到水的数量。
为什么呢？
仔细想一下，通过递归的方式我们拿三个空水瓶换一瓶水，那么此时自己手里会剩下一个空水瓶（还没喝）。那么我能不能不要空水瓶了，给老板说，我拿两个空水瓶直接换一瓶水（不要瓶子，只要里面的水）？那么的话直接两空瓶换一瓶水，并且换完后手里不再积累新的空瓶。
这样取巧的话，代码就简单很多：

import java.util.Scanner;
public class Main {
    private static Scanner in = new Scanner(System.in);
    public static void main(String[] args) {
        int c = in.nextInt();
        while (c != 0){
           int value = c/2;
           System.out.println(value);
            c = in.nextInt();
        }
    }
}
