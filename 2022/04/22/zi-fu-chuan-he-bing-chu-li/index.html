#字符串合并处理#
字符串 排序 算法
技能
发布日期:   2022-04-22
作者:   Gure
文章字数:   866
阅读时长:   3 分
阅读次数:  

描述

按照指定规则对输入的字符串进行处理。
详细描述：
第一步：将输入的两个字符串str1和str2进行前后合并。如给定字符串 “dec” 和字符串 “fab” ， 合并后生成的字符串为 “decfab”
第二步：对合并后的字符串进行排序，要求为：下标为奇数的字符和下标为偶数的字符分别从小到大排序。这里的下标的意思是字符在字符串中的位置。注意排序后在新串中仍需要保持原来的奇偶性。例如刚刚得到的字符串“decfab”，分别对下标为偶数的字符’d’、’c’、’a’和下标为奇数的字符’e’、’f’、’b’进行排序（生成 ‘a’、’c’、’d’ 和 ‘b’ 、’e’ 、’f’），再依次分别放回原串中的偶数位和奇数位，新字符串变为“abcedf”
第三步：对排序后的字符串中的’0’~‘9’、’A’~‘F’和’a’~‘f’字符，需要进行转换操作。
转换规则如下：
对以上需要进行转换的字符所代表的十六进制用二进制表示并倒序，然后再转换成对应的十六进制大写字符（注：字符 a~f 的十六进制对应十进制的10~15，大写同理）。
如字符 ‘4’，其二进制为 0100 ，则翻转后为 0010 ，也就是 2 。转换后的字符为 ‘2’。
如字符 ‘7’，其二进制为 0111 ，则翻转后为 1110 ，对应的十进制是14，转换为十六进制的大写字母为 ‘E’。
如字符 ‘C’，代表的十进制是 12 ，其二进制为 1100 ，则翻转后为 0011，也就是3。转换后的字符是 ‘3’。
根据这个转换规则，由第二步生成的字符串 “abcedf” 转换后会生成字符串 “5D37BF”。
数据范围：输入的字符串长度满足 1≤n≤100

华为机试题（HJ30 字符串合并处理）

思路

字符串合并处理过程并不复杂，按照描述的步骤进行实现即可。在转换步骤中，我们可以实现映射表进行数据映射，因为时16进制，我们创建16个数值的映射关系：0->0;1->8;2->4。。。。f->f。转换时对数据进行相关映射寻址即可。
代码如下：

import java.util.Collections;
import java.util.LinkedList;
import java.util.Scanner;
public class Main {
    // 映射表
    private static char[] letter = {'5','D','3','B','7','F'};
    private static char[] number = {'0','8','4','C','2','A','6','E','1','9'};
    private static Scanner in = new Scanner(System.in);
    public static void main(String[] args) {
        String input = in.nextLine();
        System.out.println(handleData(input));
    }

    //处理数据
    private static String handleData(String input) {
        int i = 1;
        LinkedList<Character> odd = new LinkedList<>(); //奇数位数据
        LinkedList<Character> even = new LinkedList<>(); //偶数位数据
        String[] words = input.split(" ");
        for(String word:words) {
            for (char ch : word.toCharArray()) {
                if ((i++ % 2 == 0)) {
                    even.add(ch);
                } else {
                    odd.add(ch);
                }
            }
        }
        //数据分别排序
        Collections.sort(odd);
        Collections.sort(even);
        //转换并输出
        StringBuilder sb = new StringBuilder();
        if(odd.size() == even.size()) {
            while (odd.size() > 0) {
                sb.append(getChar(odd.remove(0))).append(getChar(even.remove(0)));
            }
        } else {
            while (even.size() > 0) {
                sb.append(getChar(odd.remove(0))).append(getChar(even.remove(0)));
            }
            sb.append(getChar(odd.remove(0)));
        }
        return sb.toString();
    }
    //字符映射转换
    private static char getChar(char ch){
        if(ch >= '0' && ch <= '9'){
            return number[ch - '0'];
        } else if (ch >= 'a' && ch <= 'f'){
            return letter[ch - 'a'];
        }else if ((ch >= 'A' && ch <= 'F')){
            return letter[ch - 'A'];
        }
        return ch;
    }
}
