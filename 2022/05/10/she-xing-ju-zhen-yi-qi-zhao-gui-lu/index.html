#蛇形矩阵#一起找规律
数组 规律 算法
技能
发布日期:   2022-05-10
作者:   Gure
文章字数:   659
阅读时长:   2 分
阅读次数:  

描述

蛇形矩阵是由1开始的自然数依次排列成的一个矩阵上三角形。
例如，当输入5时，应该输出的三角形为：
1 3 6 10 15
2 5 9 14
4 8 13
7 12
11
输入描述：
输入正整数N（N不大于100）
输出描述：
输出一个N行的蛇形矩阵。

华为机试题（HJ35 蛇形矩阵）

思路一

找规律，通过描述中的样例我们发现：
第二行第n列结果为：第一行n+1列结果减一。
依次类推，只要知道了第一行的所有结果，那么就可以计算出第二行；知道第二行，就可以计算出第三行……，
根据题意，我们知道输入的数字小于100，所以我们可以计算出当最大输入N为100时，第0行的101个数据，然后依次计算出其他行的数据。
代码如下：

import java.util.Scanner;
public class Main {
    private static Scanner in = new Scanner(System.in);
    public static void main(String[] args) {
        while (in.hasNext()) {
            int input = in.nextInt();
            handleData(input);
        }
    }
    //处理数据
    private static void handleData(int input) {
        //第0行数据，依次计算出其他行数据
        int[] line0 = {0,2,4,7,11,16,22,29,37,46,56,67,79,92,106,121,137,154,172,191,
                       211,232,254,277,301,326,352,379,407,436,466,497,529,562,596,
                       631,667,704,742,781,821,862,904,947,991,1036,1082,1129,1177,
                       1226,1276,1327,1379,1432,1486,1541,1597,1654,1712,1771,1831,
                       1892,1954,2017,2081,2146,2212,2279,2347,2416,2486,2557,2629,
                       2702,2776,2851,2927,3004,3082,3161,3241,3322,3404,3487,3571,
                       3656,3742,3829,3917,4006,4096,4187,4279,4372,4466,4561,4657,
                       4754,4852,4951,5051};
        for(int y = 0;y < input;y++){
            for(int x = 0;x < input - y;x++) {
                System.out.print((line0[x+y+1]-y-1) + " ");
            }
            System.out.println();
        }
    }
}

思路二

继续找规律，这个比较厉害，我感觉可以通过行、列索引找出规律，但是研究了好大一会没找到。最终还是参考了其他大神的思路，发现：
第0行，j=1，每一位数字为：(jj+j)/2；
第1行，j=2，每一位数字为：(jj+j)/2-1.即为，去掉上一行的第一个数，然后后面的数补上来，并且减1；
从而实现代码如下：

import java.util.Scanner;
public class Main {
    public static void main(String[] args) {
        Scanner sc = new Scanner(System.in);
        while(sc.hasNext()) {
            int num = sc.nextInt();
            for (int i = 0; i < num; i++) {
                for (int j = (i + 1); j <= num; j++) {
                    System.out.print(((j*j + j) / 2) - i + " ");
                }
                System.out.println("");
            }
        }
    }
}

运行结果：

提交结果：答案正确 运行时间：97ms 占用内存：12652KB 使用语言：Java 用例通过率：100.00%
