语句和表达式

语句

语句是Python解释器可以执行的命令。Python语句中一般以新行作为语句的结束符，同时可以在同一行中使用多条语句，语句间用分号;分割。
函数之间或类的方法之间用空行分隔，表示一段新的代码的开始。类和函数入口之间也用一行空行分隔，以突出函数入口的开始。
注意：print方法默认输出是换行，如果要实现不换行需要在变量末尾加上逗号,。

表达式

表达式是由数字、算符、数字分组符号（括号）、自由变量和约束变量等以能求得数值的有意义排列方法所得的组合。单独的一个运算对象（常量/变量）也可以叫表达式，它是最简单的表达式。
表达式具有完整意义的计算机指令。
总结以上内容，我们知道：
变量是计算机运算最简单的单元。变量与运算符的有效组合就是一个表达式。一个或多个表达式构成一条可以执行的语句。

注释

python中单行注释采用#开头。
python 中多行注释使用三个单引号'''或三个双引号"""。

# 第一个注释
print ("Hello, Python!")  # 第二个注释

'''
这是多行注释，使用单引号。
这是多行注释，使用单引号。
'''

"""
这是多行注释，使用双引号。
这是多行注释，使用双引号。
"""

条件语句

条件语句是通过一条或多条语句的执行结果（True或者False）来决定执行的代码块。通过if...else...的形式控制程序的执行：

if 判断条件1 and 判断条件2：
    执行语句1……
elif 判断条件3 or 判断条件4:
    执行语句2……
else ：
    执行语句3……

当if判断条件成立时，则执行后面的语句。执行内容可以是一行或多行，使用缩进来区分同一范围。else为可选语句，当条件不成立时执行else内容。当存在多个条件时，可使用elif 条件进行多次判断。
与其他高级语言比起来，Python不支持switch语句，因此当存在多个判断条件，也需要使用elif的方式进行多次判断。

循环语句

循环语句允许程序循环、多次执行相同的语句或语句组。Python支持for循环和 while 循环两种循环语句。
循环语句中支持使用循环控制语句更改语句执行的顺序。主要包括break、continue、pass。

while 循环语句

while语句和其他高级语言方式类似，但有一点不同：Python不支持 do..while循环语句。
while支持在循环条正常执行完成时执行 else 语句块。注意当循环被break打断时，不会执行else语句。

#!/usr/bin/python
 
count = 0
while count < 5:
   print count, " is  less than 5"
   count = count + 1
else:
   print count, " is not less than 5"
for循环语句

for循环可以直接遍历任何序列的项目，如一个列表或者一个字符串。也可以使用索引的方式进行循环遍历。

# 第一个实例 
for letter in 'Python':     
   print("当前字母: %s" % letter)

# 第二个实例
fruits = ['banana', 'apple',  'mango']
for index in range(len(fruits)):
   print ('当前水果 : %s' % fruits[index])
print ("Good bye!")

for支持在循环条正常执行完成时执行 else 语句块。注意当循环被break打断时，不会执行else语句。与while语句相同。

嵌套循环

与其他高级语言类似，Python中循环语句可以相互嵌套，即一个循环中嵌套另一个循环，实现数据的多次循环。while循环与for循环可以相互嵌套。

循环控制语句

break、continue语句和其他高级语言类似，用来打破循环执行的顺序。
break语句用来终止循环语句，即循环条件没有False或者序列还没被完全递归完，也会停止执行循环语句。
continue 语句用来告诉Python跳过当前循环的剩余语句，重新继续进行下一轮循环。continue 语句跳出本次循环，而break跳出整个循环。
与其他高级语言不同的是，Python还支持 pass语句，pass语句是一个空语句，它不做任何事情，一般用做占位语句，目的是为了保持程序结构的完整性。

Python的语法是需要严格对齐的，代码更规整，可读性更好。但也引入一些问题，比如需要块语句写全。当出现没写完的块时，编译器就会报错。如果这时还不想实现全部代码，为了让编译通过，就需要用到pass。代码执行中，pass和空白行效果一样。

同时当执行无限循环语句时，可以使用 CTRL+C 来中断循环。
