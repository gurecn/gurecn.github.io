Python学习笔记（一）

Python
发布日期:   2022-12-01

文章字数:   2.9k
阅读时长:   10 分

最近在分析一些开源项目时，发现有很大一部分开源项目使用Python语言开发，包括后台使用Python实现，客户端使用Python编译动态链接库使用。比如：Athena项目、百度的飞桨等。
首先是为了研究相关的开源项目，增强公司产品的功能和体验。同时也是扩充一下Python相关知识，扩充更多的开发思路。并且重新温习一下技术开发的基本思维和基本常识。所以开始接触了解Python语言。
学习笔记主要是将学习的主要资料、文档、书籍中的重点部分进行抄录，作为学习笔记。主要参考了官方文档、菜鸟教程、《Python学习笔记–皮大庆》、《简明 Python 教程》、《Python编程：从入门到实践》等资料。

前言

Python是一种免费的、解释型的、可移植的、开放源代码的脚本编程语言。Python提供大量的模块，不仅加快了程序开发速度和代码的清晰程度，而且使程序员专注于要解决的问题，不会陷入繁琐的技术细节。它可以用来开发各种应用程序，从简单的脚本任务到复杂的、面向对象的应用程序。
语言特点：

  • 解释的；
  • 面向对象的；
  • 内建的高级数据结构；
  • 支持模块和包；
  • 支持多种平台；
  • 可扩展。

程序

程序和语言

程序是根据语言提供的指令，按照一定的逻辑顺序，对获得的数据进行运算，并将结果最终返回给我们的指令和数据的组合。程序是用编程语言写成。编程语言分高级语言和低级语言。低级语言，也叫机器语言或汇编语言。

  • 机器语言：看来是一串0和1组成的二进制数字，这些数字代表指令和数据。
  • 汇编语言：用英文单词或单词的缩写代表计算机执行的指令，使编程的效率和程序的可读性都有了较大的提高。
  • 高级语言：是人类逻辑思维的程序化、数字化和精确化数学描述。

高级语言的设计的目的是让程序按照人类的思维和语言习惯书写，是面向人的，不是面向机器的。用高级语言写的程序，必须经过“翻译”程序的处理，将其转换成机器可执行的代码，才能运行在计算机上。
高级语言翻译成机器代码有两种方法：解释和编译。解释型语言是边读源程序边执行；编译型语言则是将源代码编译成目标代码后执行。以后在执行时就不需要编译了。
自然语言：是人们日常生活中用于交流的语言，是人和人之间交流的工具。
程序语言：是人类根据自然语言的一小部分（比如英语），给计算机设计，用于人和计算机进行交流的语言。计算机按照程序步骤，分毫不差地调用自己的硬件资源进行运算，并把工作结果反馈。从这方面说，程序应该是工作流程，而非语言。
程序语言和自然语言存在很多区别：非模糊性、少冗余、无修饰。

第一个程序

Python是解释型语言。可以通过命令行或脚本模式执行Python程序。
关于Python环境的搭建在这里不做赘述，可以参考菜鸟教程相关介绍，我们这里直接进入主题。
安装好Python环境后，我们就可以直接在控制台执行Python语句或者执行脚本。

控制台执行：
PS C:\Users\Administrator\Desktop> python
Python 3.10.7 (tags/v3.10.7:6cc6b13, Sep  5 2022, 14:08:36) [MSC v.1933 64 bit (AMD64)] on win32
Type "help", "copyright", "credits" or "license" for more information.
    >>> print ("你好世界！")
    你好世界！
    >>>
通过脚本执行：

我们首先借助于文本编辑器创建一个脚本，命名为HelloWorld.py，并编辑内容如下：

print ("你好，世界")

打开控制台执行： python .\HelloWorld.py，会看到控制台输出结果：

PS C:\Users\Administrator\Desktop> python .\helloworld.py
你好，世界

此时第一个程序就执行完成了。
注意：
Python对缩进非常敏感。最常见的情况是tab和空格的混用会导致错误，或者缩进不对，而这是用肉眼无法分别的。
如果在编译时报错：IndentationError:expected an indented block，说明此处需要缩进，需要在出现错误的那一行使用空格或Tab键缩进。同时需要注意，空格或Tab键不能混用。

变量

什么是变量

变量是指向各种类型值的名字，以后再用到某个值时，直接引用这个名字即可，不用在写具体的值。这就意味着在创建变量时会在内存中开辟一个空间。Python中，变量的使用环境非常宽松。不需要类型声明，而且类型不是固定的。你可以把一个整数赋值给变量，如果觉得不合适，把字符串赋值给它完全可以（Python会自动判断赋值类型，当数据类型变更时，Python会自动创建并重新指向新的对象）。每个变量在使用前都必须赋值，变量赋值以后该变量才会被创建。等号 = 用来给变量赋值。
基于变量的数据类型，解释器会分配指定内存，并决定什么数据可以被存储在内存中。因此，变量可以指定不同的数据类型，这些变量可以存储整数，小数或字符。

变量名规范

变量名只能包含字母、数字和下划线。变量名能以字母或下划线开头，但不能以数字开头。Python中变量名是区分大小写的，Name和name是两个不同的变量名。
变量名不能包含空格，可使用下划线来分隔单词。
不能使用Python中系统函数名、保留字和关键字变量命名：and、continue、else、for、import、not、raise、assert、def、except、from、in、or、return、break、del、exec、global、is、pass、try、class、elif、finally、if、lambda、print、while
变量名应既简短又具有意义。name 比 n 好；name_length 比 length_of_name 好。
慎用小写字母l和字母O，因为可能被错看成数字1和数字0。

数据类型

Python有五个标准的数据类型：

  • Numbers（数字）：int（有符号整型）、long（长整型）、float（浮点型）、complex（复数）。
  • String（字符串）：由数字、字母、下划线组成的一串字符，支持单引号和双引号。
  • List（列表）：列表可以完成大多数集合类的数据结构实现。它支持字符，数字，字符串甚至可以包含列表（即嵌套）。列表用 [ ] 标识。
  • Tuple（元组）：元组类似于 List（列表）。用 () 标识，内部元素用逗号隔开。但元组不能二次赋值，相当于只读列表。
  • Dictionary（字典）：类似Map。列表是有序的对象集合，字典是无序的对象集合。字典中的元素通过键来存取。字典用”{ }”标识。

布尔

逻辑判断的是或非，值可取True和False。
布尔类型数据支持逻辑运算，包括与（and、一假则假）、或（or、一真则真）、非（not、真假倒转）。
注意：逻辑运算符还可以用于数字运算，在数字运算中，0和False被认为是布尔类型中的False；其他数值和True被认为是布尔类型中的True。所以依据逻辑分析，a and b中由前向后，遇到a为布尔类型的False（0和False）则返回结果为a的值，否则返回b的值。a or b类似，遇到a为布尔值中的True（其他数值和True）时返回a的值，否则返回b的值。

数字

数字类型支持常见的算数运算，包括加（+）、减（-）、乘（*）、除（/）、取模（%，除法的余数）、取整（//，除法的商）幂运算（**）等。
数字类型支持常见的比较运算，包括等于（==）、不等于（!=）、大于（>）、小于（<）、大于等于(>=)、小于等于(<=)等。
数字类型支持常见的赋值运算，包括赋值（=）、加法赋值（+=）、减法赋值（-=）、乘法赋值（*=）、除法赋值（/=）、取模赋值（%=）、幂运算赋值（**=）、取整赋值（//=）。
Python支持位运算，包括按位与（&）、按位或（|）、按位异或（^）、按位取反(~)、左移(<<)、右移(>>)等。
注意：

long 类型只存在于 Python2.X 版本中。在 2.2 以后的版本中，int 类型数据溢出后会自动转为long类型。在 Python3.X 版本中 long 类型被移除，使用 int 替代。
长整型也可用小写 l，但建议您使用大写 L，避免与数字 1 混淆。Python使用 L 来显示长整型。
Python 还支持复数，复数由实数部分和虚数部分构成，可以用 a + bj,或者 complex(a,b) 表示， 复数的实部 a 和虚部 b 都是浮点型。

字符串

#!/usr/bin/python
# -*- coding: UTF-8 -*-
 
str = 'Hello World!'
 
print str           # 输出完整字符串
print str[0]        # 输出字符串中的第一个字符
print str[2:5]      # 输出字符串中第三个至第六个之间的字符串
print str[2:5:2]    # 输出字符串中第三个至第六个之间的字符串,截取步长为2
print str[2:]       # 输出从第三个字符开始的字符串
print str * 2       # 输出字符串两次
print str + "TEST"  # 输出连接的字符串

以上运算输出以下结果：

Hello World!
H
llo
lo
llo World!
Hello World!Hello World!
Hello World!TEST

针对字符串，列表或元组，支持成员运算。包含（in）、不包含（not in）。

列表

列表使用方括号 [ ] ，元组使用小括号 ( ) 。

list = [ 'runoob', 786 , 2.23, 'john', 70.2 ]
tinylist = [123, 'john']
 
print list               # 输出完整列表
print list[0]            # 输出列表的第一个元素
print list[1:3]          # 输出第二个至第三个元素 
print list[2:]           # 输出从第三个开始至列表末尾的所有元素
print tinylist * 2       # 输出列表两次
print list + tinylist    # 打印组合的列表

输出为：

[‘runoob’, 786, 2.23, ‘john’, 70.2]
runoob
[786, 2.23]
[2.23, ‘john’, 70.2]
[123, ‘john’, 123, ‘john’]
[‘runoob’, 786, 2.23, ‘john’, 70.2, 123, ‘john’]

元组

元组的取值操作和列表相同，但是元组无法对指定位置进行二次赋值。下面操作是非法的：

tuple = ( 'runoob', 786 , 2.23, 'john', 70.2 )
tuple[2] = 1000    # 元组中是非法应用

执行会报错：TypeError: 'tuple' object does not support item assignment。

字典

字典由索引(key)和它对应的值value组成。
下面为对字典的赋值和操作：

dict = {}
dict['one'] = "This is one"
dict[2] = "This is two"
tinydict = {'name': 'runoob','code':6734, 'dept': 'sales'}
print dict['one']          # 输出键为'one' 的值
print dict[2]              # 输出键为 2 的值
print tinydict             # 输出完整的字典
print tinydict.keys()      # 输出所有键
print tinydict.values()    # 输出所有值

输出结果为：

This is one
This is two
{‘dept’: ‘sales’, ‘code’: 6734, ‘name’: ‘runoob’}
[‘dept’, ‘code’, ‘name’]
[‘sales’, 6734, ‘runoob’]

对象

针对对象数据，支持身份运算。是（is，两个标识为同一对象）、不是（is not，两个标识为不同对象）。
is 用于判断两个变量引用对象是否为同一个(同一块内存空间)， == 用于判断引用变量的值是否相等。
